# OpenID Connect API
This is a documentation page for the OpenID Connect API provided by Haste Health. OpenID Connect is an identity layer built on top of the OAuth 2.0 protocol, allowing clients to verify the identity of end-users based on the authentication performed by an authorization server.

* [Token Endpoint](/docs/API/OpenID%20Connect/Endpoints/token)
   Used to obtain tokens (ID tokens, access tokens, refresh tokens) after the user has authenticated.
* [Authorize Endpoint](/docs/API/OpenID%20Connect/Endpoints/authorize)
   Used to initiate the authentication process and obtain authorization from the user.
* [WellKnown Endpoint](/docs/API/OpenID%20Connect/Endpoints/well-known)
   Provides metadata about the OpenID Connect provider, including supported scopes, response types, and endpoints.
